Tampermonkey

You can use a tampermonkey script to upload the test directly from moodle.
To use the script do the following:
  1. Download the tampermonkey extension if you haven't already. Available on firefox or Chrome.
  2. Click this link to download the script.
  3. Open a test in moodle. If the script is active, a few buttons in the bottom left corner should have appeared.
  4. Add your api-key (by pressing the corresponding button). You can create an api-key in your tsync account settings.
  5. If everything worked, pressing the update button in moodle should now update the test in tsync.
