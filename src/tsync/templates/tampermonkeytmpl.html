Tampermonkey

Setup Instructions

  1. Download Tampermonkey extension (Firefox or Chrome).
  2. Click this link to download the script.
  3. Open a test in Moodle. If the script is active, buttons will appear in the bottom left corner.
  4. Add your API key in the script settings. You can create one in your Account page.
  5. Press the update button in Moodle to sync the test.

⚠️ Warning

If using a Chromium-based browser, you need to enable user scripts:

  1. Go to Extension Management
  2. Find Tampermonkey and click "Details"
  3. Enable "Allow user scripts"
